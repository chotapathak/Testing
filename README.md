# Intro for this Repo
learning to Integrate the smart contract with local networks and Testing
### Get it
```
git clone https://github.com/chotapathak/Testing.git
cd Testing
```
## Basic requirements
```
npm init -y 
npm i -g truffle 
npm i mocha chai --save-dev
npm i --save-dev @openzepplin/test-helpers
```
### Steps to run
```
truffle develop
truffle test
```
